Completed Behaviors

  • Connected dApp origins are persisted in extension storage instead of existing only in memory.
  • signTransaction prepares a real transaction before approval and returns real signed hex.
  • sendTransaction approves and broadcasts the same prepared transaction, then returns the real txid.
  • signMessage returns a deterministic Schnorr signature using the active Taproot account key.
  • getBalance refreshes balance from a live discovery scan when the wallet is unlocked, and falls back to cached balance only when needed.
  • The approval popup now shows executable details instead of generic placeholders.
